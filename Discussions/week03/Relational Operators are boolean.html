Correct. The relational operators (as we call them, since they test for a specific relationship between two items of data) work on any kind of data. The result will be an indication of whether that relationship exists between those two items. So, "whether or not" is either yes or no, or true or false.

So... 

1 < 3 is true
4 >= 15 is false
"cat" == "dog" is false

Java will complain if you try to compare two values that don't naturally compare to each other. 

1 < "cat" doesn't make sense, you'll get an error in your source code.
