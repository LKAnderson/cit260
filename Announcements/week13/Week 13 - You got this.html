What's due this week?

What you need to do this week is code up all of the classes that will make up your program. There is an emphasis on commenting your code well. You should identify every class and the fields and methods in each class that will be needed, but you do not need to have every method coded by the end of the week. You can have pseudocode comments in them if you don't have actual code. (Next week will be where everything gets implemented).
