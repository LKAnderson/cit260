Exam Info

By the end of this week, you will have seen everything this introductory course intends to show you. You should congratulate yourself. Great job making it this far! You'll gain mastery of it all as you practice it and get more experience with it.

Exam #2 is this week. It consists of 35 multiple choice questions and covers the material from the second half of the semester. It has a 2 hour time limit. You'll also want to know stuff from the first half of the semester, as it everything in programming builds on itself. In particular, you will want to make sure you've got a good understanding of the following topics:

  • Pass by value and pass by reference, and which types of data are passed by value and which are passed by reference.
  • How passing by reference actually works in Java.
  • The Java File class and what you can do with it.
  • Java Exceptions, and how they are thrown and caught, and what that finally clause does.
  • Arrays, ArrayLists and the types and how loops work with them.
  • As always, you'll need to be able to see what code is valid Java and what is not.

Here are some tips I have for taking the test:

  1. Don't overthink the questions. They're intended to test your understanding, but not to trick you. 
  2. You will want to keep the Javadocs from Oracle handy. They will help you answer some questions, so knowing how to find what you're looking for will save you some time. 
  3. JShell is a helpful tool to have handy, too. You'll need JDK 9 or newer in order to use it.

As with the first exam, this exam is open book, open Internet, open Eclipse/JShell/etc. But it is not open friend. You need to do your own research and answer the questions using your own understanding.

Please do not discuss test questions in the discussion boards, or in any other medium with other students in the class. If you have concerns with a question, you may contact me directly via e-mail (andersonkent@byui.edu).

Good luck! You've all got this!
