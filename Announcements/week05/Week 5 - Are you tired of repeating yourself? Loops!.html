Three types of Loops

There are just three basic types of loops. And they're all very similar to each other.

The While Loop

        // set up starting condition
            boolean condition = true;
    
            // loop until something sets the condition
            // to false
            while (condition == true) {
    
                // Your repeated code goes in here.
    
                // Something needs to happen to
                // set your condition to false
                // so your loop can eventually
                // exit
                condition = false;
            }

The Do-While Loop

        // It's the same, except the body of the
            // loop is executed at least one time
            // before the condition is checked.
            boolean condition = true;
    
            do {
                // Your repeated code goes in here.
    
                // Something needs to happen to
                // set your condition to false
                // so your loop can eventually
                // exit
                condition = false;
    
            } while (condition == true);

The For-Loop

        // For loop - this is just a 
            // convenient shorthand of the while
            // loop.
            
            int counter = 0;
            while (counter < 10) {
                
                // Do repeated stuff
                
                counter += 1;
            }
            
            // We can re-write this while loop like this...
            
            for (int counter = 0; counter < 10; counter += 1) {
                
                // Do repeated stuff
            }

That's all the loops we have. Well, ok, there's one more we'll get to, called a for-each loop, but it's even easier than a regular for loop, and we'll get to it in Chapter 7 when we get to Arrays.
